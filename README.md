# ManagementUser

Projeto utilizado na disciplina de Manutenção e Evolução de Software.

## Objetivo

Construir uma API REST para gerenciamento de usuários e perfis utilizando ASP.NET Core.

## Tecnologias

* C#
* ASP.NET Core 8
* Entity Framework Core
* MySQL
* Swagger
* Git
* GitHub

## Pré-requisitos

* Visual Studio 2022
* .NET 8 SDK

## Como executar

Clone o repositório:

git clone URL_DO_REPOSITORIO

Acesse a pasta do projeto:

cd ManagementUser

Compile:

dotnet build

Execute:

dotnet run

Acesse o Swagger:

[http://localhost:xxxx/swagger](http://localhost:xxxx/swagger)
